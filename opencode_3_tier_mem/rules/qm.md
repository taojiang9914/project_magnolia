# QM Calculation Rules

## Mandatory: Input File Requirements

Every QM calculation must have:

```
work_dir/
├── molecule.xyz       # Cartesian coordinates
├── molecule.inp       # ORCA input (generated by orca_setup)
├── molecule.com       # Gaussian input (generated by gaussian_setup)
└── molecule.out/.log  # Output from completed calculation
```

- Coordinates must be valid XYZ or PDB format
- Charge and multiplicity must be defined before running
- Input files should be generated via `orca_setup` or `gaussian_setup`, not manually

## ORCA Input Generation

```
orca_setup(input_file, method, basis, charge, multiplicity, task, solvent, output_dir, ncores)
```

### Common Method/Basis Combinations

| Purpose | Method | Basis |
|---------|--------|-------|
| Quick screening | B3LYP | def2-SVP |
| Production SP | B3LYP | def2-TZVP |
| High accuracy | wB97X-D | def2-TZVPP |
| Dispersion | B3LYP-D3BJ | def2-TZVP |
| Transition metals | TPSSh | def2-TZVP |

### Task Keywords

| Task | ORCA Keyword | Description |
|------|-------------|-------------|
| SP | (none) | Single-point energy |
| OPT | OPT | Geometry optimization |
| FREQ | FREQ | Frequency calculation |
| OPTFREQ | OPT FREQ | Optimization + frequencies |

### Solvent Effects

Use CPCM implicit solvation: `solvent="water"`, `solvent="dmso"`, etc.

## Gaussian Input Generation

```
gaussian_setup(input_file, method, basis, charge, multiplicity, task, output_dir)
```

### Common Method/Basis Combinations

| Purpose | Method | Basis |
|---------|--------|-------|
| Quick SP | B3LYP | 6-31G* |
| Production | B3LYP | 6-311+G** |
| High accuracy | M06-2X | aug-cc-pVTZ |
| Transition metals | B3LYP | LANL2DZ |

## Common Mistakes

| Mistake | Correct |
|---------|---------|
| Wrong charge on ion | Verify formal charge: Na+ = +1, Cl- = -1 |
| Singlet on radical | Open-shell systems need multiplicity > 1 |
| No freq after opt | Always do OPT+FREQ to verify minimum (no imaginary freqs) |
| SCF not converging | Try tighter convergence, change guess, use DIIS |
| Wrong units | ORCA/Gaussian output energies are in Hartree |
| Missing solvent | Solution-phase work always needs implicit solvent |
| Running on head node | Use `submit_job` for Slurm/PBS submission |

## Job Management

### Submit to Scheduler

```
submit_job(command, working_dir, scheduler="slurm", job_name="qm", ncores=4, memory="4GB", time_limit="24:00:00")
```

### Check Status

```
check_job(job_id, scheduler="slurm")
```

### Cancel Job

```
cancel_job(job_id, scheduler="slurm")
```

### Supported Schedulers

- `slurm` — sbatch/squeue/scancel
- `pbs` — qsub/qstat/qdel
- `local` — background process

### Resource Guidelines

| Calculation | Cores | Memory | Time |
|-------------|-------|--------|------|
| Small molecule SP | 4 | 4GB | 1h |
| Medium optimization | 8 | 8GB | 12h |
| Large freq calculation | 16 | 16GB | 24h |
| Transition metal complex | 8-16 | 16GB | 48h |

## ORCA Parsing

`orca_parse` extracts from .out files:
- Final single-point energy (Hartree and eV)
- HOMO-LUMO gap
- Geometry convergence status
- SCF convergence status
- Orbital energies

## Gaussian Parsing

`gaussian_parse` extracts from .log files:
- SCF energies (all and final)
- HOMO and LUMO energies
- HOMO-LUMO gap
- Frequencies (including imaginary count)
- Thermal corrections (zero-point, enthalpy, Gibbs)
- Normal termination status

## Stage Gates

- `qm_inputs_defined` — checks coordinate files exist, charge/multiplicity defined
- `scf_converged` — checks QM output shows SCF convergence

## Error Resolution

| Error | Fix |
|-------|-----|
| SCF not converging | Increase MaxIter, try LevelShift, change initial guess |
| Imaginary frequencies | Not at minimum; re-optimize with tighter criteria |
| ORCA segfault | Reduce ncores or memory per core |
| Gaussian l103 error | Bad initial geometry; pre-optimize with xTB |
| `orca` not found | Install ORCA and add to PATH |
| `g16` not found | Install Gaussian and add to PATH |
| `sbatch` not found | Use `scheduler="local"` for local execution |

## Workflow Template

```
1. orca_setup(xyz_file, method, basis, charge, mult, task="OPT")
2. submit_job("orca molecule.inp", work_dir, scheduler="slurm")
3. check_job(job_id) until completed
4. orca_parse("molecule.out")
5. stage_gate("scf_converged", work_dir)
```
